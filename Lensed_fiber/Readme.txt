Data & data analysis for "Ultra-narrow linewidth laser across the C-band using polarization-controlled dual-cavity feedback".

Data is saved in the 'Data' zip folder.

Data analysis is carried out in Python notebooks in 'RIN_analysis.ipynb, 'Linewidth_analysis.ipynb', 
'Rayleigh_backscatter_analysis', and 'FN_analysis.ipynb'.

The remaining Python files are loaded into and used in the notebooks.
